# AI-CBT-Assistant
A conversational AI application designed to support CBT-inspired coaching and reflective conversations through a Telegram interface. The system integrates large language models, conversation memory, context summarization, speech-to-text transcription, and persistent user session management to enable coherent long-form interactions.

The project explores practical challenges in deploying conversational AI systems, including memory management, context window limitations, conversation summarization, user state persistence, and real-world interaction workflows.
